⏺ No — Keywords Everywhere has no MCP interface and no public API. It's a pure browser extension
  (Chrome/Firefox) that injects volume data directly onto Google search result pages. The data    
  pipeline is: you search Google → the extension calls their API in the background → volume/CPC
  appears inline. There's no way to programmatically query it outside the browser.
